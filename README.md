A full-stack MERN doctor appointment booking platform where patients can book appointments, doctors manage schedules & earnings, and admins control the system.

Built using MongoDB, Express.js, React.js, Node.js, and Tailwind CSS.

This project is inspired by a tutorial and rebuilt as a resume-ready full stack application.

--------------------------------------------------
Features

Patient Panel
- Register & login
- Browse doctors
- Book appointments
- Cancel appointments
- Manage profile
- Online payment integration

Doctor Panel
- Doctor dashboard
- View appointments
- Track earnings
- Update profile

Admin Panel
- Admin authentication
- Add & manage doctors
- View all appointments
- Dashboard analytics
